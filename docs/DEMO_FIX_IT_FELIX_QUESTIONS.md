# Fix it Felix Demo Questions

Use these Cummins-focused prompts for live demos.

## Core Diagnostics

1. Diagnose fault code `SPN 157 FMI 18` on a Cummins ISX15.
2. What are common failure symptoms of a failing fuel injector on ISX15?
3. How do I confirm whether this is wiring vs injector vs ECM issue?
4. Give me a step-by-step injector diagnostic checklist for rough idle and white smoke.

## Repair Workflow

1. What tools and safety checks are required before injector replacement?
2. Create a repair checklist for coolant leak and overheating under load.
3. What parts are typically needed for turbocharger replacement on ISX platform?
4. What should I verify after repair before closing the ticket?

## Ticket + Ops Automation

1. Summarize this ticket and suggest technician handoff notes.
2. Generate a triage plan with estimated effort and risk level.
3. Based on this issue description, what questions should I ask the technician next?
4. Turn this chat into a structured service ticket draft.

## Best Demo Prompt Format

For stronger outputs, include:

- Engine model
- Fault code(s)
- Symptoms
- Operating conditions
- What has already been tried

Example:

`Cummins ISX15. Fault code SPN 157 FMI 18. Rough idle + white smoke under load, intermittent power loss at high RPM. Fuel filter replaced, issue persists. Give me triage steps and likely root causes.`

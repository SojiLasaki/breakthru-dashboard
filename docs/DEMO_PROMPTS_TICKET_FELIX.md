# Demo Prompts for Ask Fix-it Felix (Ticket Context)

## Getting Started

Navigate to any ticket detail page (e.g., `/tickets/1`) and click the **sparkle button** (bottom-right) or the **"Ask Fix-it Felix"** button in the full detail view.

---

## Thread Memory Demo

Send this first:
```
What's the fault code for this ticket?
```

Then follow up with:
```
What usually causes that?
```

Then:
```
How long does this repair typically take?
```

*Notice: Felix remembers the context from previous messages!*

---

## Pro Tips for Demo

1. **Show Thread Memory**: Close the panel, navigate away, come back - the conversation persists!

2. **Show Context Awareness**: Ask "summarize this ticket" and Felix will reference the actual ticket data

3. **Show Suggestion Chips**: Click the quick suggestions like "What should I check first?" to auto-fill

4. **Show Full Detail View**: Click "View Full Repair Details & Checklist" and use the header "Ask Fix-it Felix" button

## Feature Highlights

| Feature | Description |
|---------|-------------|
| **Floating Button** | Sparkle icon in bottom-right corner of ticket summary view |
| **Header Button** | "Ask Fix-it Felix" button in full detail view header |
| **Thread Memory** | Conversations persist per-ticket across sessions |
| **Auto Context** | Ticket ID, diagnostic info, and checklist state auto-injected |
| **Suggestion Chips** | Quick-start prompts for common questions |
| **Checklist Updates** | Felix can propose checklist changes (approve with one click) |
